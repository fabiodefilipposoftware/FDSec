# Here there will be blacklist and whitelist hashes and signatures dataset

## HASHES
You can add malware sha256 hashes from your analysis or other resources (example, Virustotal);

## SIGNATURES
You can add hexadecimal signatures with \x or blank/backspace, for example <strong>\xa1\xxb2\xc3</strong> or <strong>a1 b2 c3</strong>;

You can write conditions, for example <strong>(\xa1\xxb2\xc3 AND \xd4\xe5\xf6) OR (\xb1\xc2\xd3 AND \xe4\xf5\xa6)</strong>;
